# 你是哪種工作風格？
> A personality quiz about workplace styles — built with vanilla HTML, CSS & JavaScript.

🔗 **[點我開始測驗](https://soft-manto.github.io/work-style-quiz/)**

---

## 關於這個專案

這是我第一個從零開始、搭配 AI 工具完成的前端小專案。

透過 8 道情境題，測出你在職場中最自然的行為模式，共有五種結果類型：

| 類型 | 特色 |
|------|------|
| 🧠 策略家 | 規劃導向、系統思考、長遠視野 |
| ⚡ 執行者 | 行動派、效率優先、結果導向 |
| 🤝 協調者 | 溝通型、重視團隊、善於建立共識 |
| 🎨 創意人 | 跳躍思考、不拘框架、點子王 |
| 🔍 分析師 | 數據導向、謹慎細心、邏輯清晰 |
